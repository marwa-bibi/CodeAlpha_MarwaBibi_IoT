# Task 2: Sensor-Based Simulation

## Overview
This project demonstrates an automatic lighting system using an LDR (Light Dependent Resistor) and Arduino Uno. The system detects ambient light intensity and automatically controls an LED based on predefined threshold values.

## Components Used
- Arduino Uno
- LDR Sensor
- LED
- Resistors
- Breadboard

## Software Used
- Tinkercad
- Arduino IDE

## Project Features
- Automatic LED ON/OFF
- Light Intensity Detection
- Sensor-Based Automation
- Arduino Programming
